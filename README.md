# MiMo Agent Code Reader

一个面向学生开发者、课程项目和科研项目入门场景的代码仓库理解与工程任务拆解 Agent 原型。

## 项目背景

学生开发者在接触陌生课程项目、开源项目、科研代码或机器人软件项目时，经常会遇到以下问题：

- 不知道如何快速理解项目目录结构
- 不清楚依赖环境如何配置
- 遇到 Python / Node.js / Git / npm / 代理环境报错时，不知道从哪里排查
- 难以把代码理解、调试过程和实验记录整理成文档
- 缺少适合中文工程语境的 AI Agent 工作流

本项目希望基于 Xiaomi MiMo API，结合 OpenCode / Codex / Claude Code 类工具链，构建一个面向学生开发者的代码仓库理解与工程任务拆解 Agent。

## 核心目标

- 自动分析项目目录结构
- 解释核心代码模块和文件职责
- 生成运行环境检查清单
- 输出依赖安装与报错排查步骤
- 辅助生成 README、实验报告、学习笔记和开发日志
- 测试 MiMo 在中文代码解释、长上下文理解、多轮命令规划中的表现

## 典型使用场景

1. 学生接手陌生课程项目时，快速理解项目结构和运行方式；
2. 配置 Python / Node.js / Git / npm / 代理环境时，自动生成排查路径；
3. 阅读科研项目或机器人软件项目时，拆解模块职责和运行流程；
4. 对论文文档、实验报告、项目说明进行格式检查和内容整理；
5. 对比 MiMo、GPT、Claude、Gemini、DeepSeek 在学生开发场景中的表现。

## 计划接入能力
